

Short description of experience
===============================

The Hadoop part of the assignment was stepwise, granular and clear.  It is good to 
see how machines are set up for Hadoop, added to the cluster and the HDFS file system. 

It is unclear how nodes can be added after the initial cluster has been set up. 

The Terasort part of the homework is opaque. Not clear how the Map or Reduce processes are kicking in and 
what they are each doing.  The utility of Hadoop native libs versus the Java libs is also not clear. How impactful 
the warning with the Java libs is not clear. 

The functioning of YARN and the Resource Manager will also be nice to observe.  Debug hooks will be nice to have. 
In order to observe traffic flow during map and reduce.
